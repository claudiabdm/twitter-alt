# twitter-alt


## Description
Firefox extension to show image alt text.

## Features
- Show alt text from images when hovering on them.

## Build Setup

```bash
# install web-ext
$ npm install --global web-ext

# open dev environment
$ web-ext run

# build for production
$ web-ext build
```
